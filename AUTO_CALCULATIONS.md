# ✅ AUTOMATIC CALCULATIONS & DEFAULT STATUS!

## 🎯 Smart Invoice Calculations

### What's New:

#### 1. **Automatic Totals Calculation**
When you add line items, the system automatically calculates:
- ✅ **Subtotal** - Sum of all line items
- ✅ **VAT (20%)** - Calculated from subtotal
- ✅ **Total Amount** - Subtotal + VAT

#### 2. **Client Invoices Default to "Pending"**
- Client invoices (Income) automatically set to **"Pending Payment"**
- Ready for payment link generation
- Can be changed to "Paid" after payment received

### How It Works:

**With Line Items:**
```
📦 Web Design - 1 × 1500.00 = 1500.00
📦 Logo Design - 1 × 500.00 = 500.00
📦 Hosting - 12 × 50.00 = 600.00

---
Subtotal:     2600.00 MAD
VAT (20%):     520.00 MAD
---
Total:        3120.00 MAD  ← Auto-calculated!
```

**Without Line Items:**
- Manual entry fields for VAT and Total
- You enter amounts yourself

### Display:

**When Line Items Exist:**
- Beautiful summary display
- Subtotal in gray
- VAT (20%) in gray
- **Total in large, bold, secondary color**
- All amounts show currency
- Read-only (auto-calculated)

**When No Line Items:**
- Editable input fields
- Manual entry mode
- For simple invoices

### Client Invoice Flow:

1. **Choose "Client Invoice"** (Income)
2. **Status automatically set to "Pending"** ✅
3. **Add line items**
4. **Totals calculate automatically** ✅
5. **Save invoice**
6. **Generate payment link** (coming soon!)
7. **Send to client**
8. **Mark as "Paid" when received**
